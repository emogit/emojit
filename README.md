# Emojit Extension

Rate any webpage.

## Install

    cd ext/
	yarn install

## Development

    cd ext/

Then one of:

    yarn dev chrome
    yarn dev firefox
    yarn dev opera
    yarn dev edge

## Build

    cd ext/

Then one of:

    yarn build chrome
    yarn build firefox
    yarn build opera
    yarn build edge

## Environment

The build tool also defines a variable named `process.env.NODE_ENV` in your scripts.
